# Исправление проблемы с кодировкой (иероглифы вместо текста)

## ❌ Проблема

При запуске `publish-to-github.bat` вместо русского текста отображаются иероглифы/кракозябры.

## ✅ Решение

### Способ 1: Обновлённый скрипт (автоматически)

Скрипт уже обновлён и включает команду `chcp 65001` для переключения на UTF-8.

**Просто запустите:**
```cmd
publish-to-github.bat
```

### Способ 2: Вручную изменить кодировку консоли

Перед запуском выполните:

```cmd
chcp 65001
publish-to-github.bat
```

Или в одной строке:
```cmd
chcp 65001 && publish-to-github.bat
```

### Способ 3: Изменить шрифт консоли

1. Откройте Command Prompt
2. Кликните правой кнопкой на заголовке окна
3. Выберите "Свойства"
4. Вкладка "Шрифт"
5. Выберите **Lucida Console** или **Consolas**
6. Нажмите OK

---

## 🔧 Для PowerShell скрипта

PowerShell скрипт также обновлён, но если возникают проблемы:

```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
.\publish-to-github.ps1
```

Или создайте профиль PowerShell:

1. Откройте PowerShell
2. Выполните:
```powershell
notepad $PROFILE
```
3. Добавьте строку:
```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```
4. Сохраните и перезапустите PowerShell

---

## 📝 Проверка текущей кодировки

```cmd
chcp
```

**Вывод:**
- `866` - OEM кодировка (проблема)
- `65001` - UTF-8 (правильно)

---

## 🎯 Быстрое исправление (все проблемы)

```cmd
@echo off
chcp 65001
cls
publish-to-github.bat
```

Сохраните как `fix-and-run.bat` и запускайте этот файл.

## ⚙️ Глобальная настройка Windows 10/11

Для постоянной поддержки UTF-8:

1. Панель управления → Язык и региональные стандарты
2. Административные языковые параметры
3. Изменить язык системы
4. ✅ Включить "Beta: Use Unicode UTF-8 for worldwide language support"
5. Перезагрузите компьютер

**⚠️ Внимание:** Это может повлиять на работу старых программ!

## 🐛 Если всё ещё проблема

### Проверьте кодировку файла

1. Откройте `publish-to-github.bat` в Блокноте
2. Файл → Сохранить как
3. Внизу выберите кодировку **UTF-8**
4. Сохраните

### Используйте PowerShell вместо CMD

```powershell
.\publish-to-github.ps1
```

PowerShell лучше работает с Unicode.

### Запустите от имени администратора

Иногда права доступа влияют на кодировку:
1. ПКМ на файле
2. "Запуск от имени администратора"
