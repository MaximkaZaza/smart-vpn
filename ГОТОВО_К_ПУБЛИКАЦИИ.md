## 🚀 3 ШАГА ДО ПУБЛИКАЦИИ

### Шаг 1: Создайте репозиторий

```
1. Откройте https://github.com/new
2. Имя: smart-vpn
3. Public или Private
4. НЕ инициализируйте README
5. Нажмите "Create repository"
```

### Шаг 2: Запустите скрипт

**Windows:**
```cmd
publish-to-github.bat
```

**Или PowerShell:**
```powershell
.\publish-to-github.ps1
```

**Или вручную:**
```bash
git init
git add -A
git commit -m "Initial commit: Smart VPN project"
git remote add origin https://github.com/YOUR_USERNAME/smart-vpn.git
git push -u origin main
```

### Шаг 3: Настройте Secrets

```
Settings → Secrets and variables → Actions

Добавьте:
- DOCKER_USERNAME
- DOCKER_PASSWORD
- TELEGRAM_BOT_TOKEN
```
