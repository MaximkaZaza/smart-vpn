# VPN Project — Контекст для Qwen Code

## Обзор проекта

Пустой проект для разработки VPN-приложения на **Node.js**. Предназначен для создания VPN-сервера с возможностью расширения функциональности.

### Технологии

- **Runtime:** Node.js
- **Язык:** JavaScript
- **Dev-инструменты:** nodemon (автоматическая перезагрузка при разработке)

### Структура проекта

```
VPN/
├── src/              # Исходный код приложения
│   └── index.js      # Точка входа (entry point)
├── config/           # Конфигурационные файлы (пусто)
├── tests/            # Тесты (пусто)
├── docs/             # Документация (пусто)
├── node_modules/     # Зависимости npm
├── .gitignore        # Игнорируемые файлы Git
├── package.json      # Конфигурация npm
├── package-lock.json # Заблокированные версии зависимостей
├── README.md         # Основная документация
└── QWEN.md           # Этот файл (контекст для AI)
```

## Сборка и запуск

### Установка зависимостей

```bash
npm install
```

### Команды npm

| Команда | Описание |
|---------|----------|
| `npm start` | Запуск в production режиме |
| `npm run dev` | Запуск в development режиме с nodemon (auto-reload) |
| `npm test` | Запуск тестов (не настроено) |

### Точка входа

Файл `src/index.js` — основной файл запуска приложения.

## Конвенции разработки

### Стиль кода

- JavaScript (без TypeScript на текущий момент)
- Стандартный стиль Node.js

### Git

Игнорируемые файлы (`.gitignore`):
- `node_modules/` — зависимости
- `.env` — переменные окружения
- `*.log` — лог-файлы
- `dist/` — скомпилированный код
- `coverage/` — отчёты тестирования
- `.DS_Store` — метаданные macOS

### Тестирование

Тесты не настроены. Директория `tests/` пуста.

## Расширение проекта

Рекомендуемые направления для развития:

1. **Конфигурация** — добавить файлы конфигурации в `config/`
2. **VPN-протоколы** — реализовать поддержку WireGuard, OpenVPN или SOCKS5
3. **Тесты** — настроить Jest или Mocha в директории `tests/`
4. **Документация** — заполнить `docs/` технической документацией
5. **Зависимости** — добавить необходимые npm-пакеты для VPN-функциональности
